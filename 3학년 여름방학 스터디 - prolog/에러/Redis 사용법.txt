Redis db 설치
https://github.com/microsoftarchive/redis/releases

설치위치
D:\Redis

Redis db 서버 실행 
(.exe파일) redis-server
(cmd 명령어) redis-server --service-start

Redis db 서버 중지
(.exe파일) ctrl + c 
(cmd 명령어) redis-server --service-stop

Redis db 명령어 실행
(.exe파일) redis-cli

gradle 의존성 추가
implementation 'org.springframework.boot:spring-boot-starter-data-redis'

application.yml redis 설정
spring:
  redis:
    host: 127.0.0.1 # localhost
    port: 6379

RedisConfig, RedisRepository 추가 -> Repository를 이용한 객체 저장 방식 (RedisTemplate 방식에 비해서 jpa와 사용이 비슷한것으로 보임)

PrologApplications.class -> @EnableCaching 어노테이션 추가

Access Token, Refresh Token을 동시에 사용함 Access Token의 경우 Refresh Token에 비해 시간이 짧음
